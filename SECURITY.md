# Security Policy

## Supported Versions

We only provide security updates for the current major version.

| Version | Supported          |
| ------- | ------------------ |
| 1.5.x   | :white_check_mark: |
| < 1.5   | :x:                |

## Reporting a Vulnerability

We take the security of the VEX protocol seriously. If you discover a vulnerability, please do not disclose it publicly. Instead, please report it privately via email to:

**security@provnai.com**

Please include:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.

We will acknowledge receipt of your report within 48 hours and work with you to resolve the issue before public disclosure.
